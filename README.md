# Cyber Security Awareness Bot

## Overview
This project is a Cyber Security Awareness Chatbot developed using C# and WPF.

The chatbot helps users learn about:
- Password safety
- Phishing attacks
- Online privacy
- Online scams

## Features
- Modern WPF GUI
- Voice greeting
- Cybersecurity topic buttons
- Keyword recognition
- Sentiment detection
- Memory feature
- Random responses
- Error handling

## Technologies Used
- C#
- WPF
- .NET 8
- Visual Studio Code
